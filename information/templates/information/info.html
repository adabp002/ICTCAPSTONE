1. Introduction

1.1 Purpose

The Sentence Requirement Classification Tool is designed to automate the categorization of security requirements based on input sentences. This user manual will guide you through the tool's features and functionalities.

1.2 System Overview

The Sentence Requirement Classification Tool is designed to streamline the process of identifying and categorizing security requirements within software development. It offers several features to enhance the user experience and improve the accuracy of requirement classification. By using this tool, users can expect the following:

1.2.1 Main Features

  • Automated Sentence Classification: The tool uses advanced machine learning models to automatically categorize security requirements based on input sentences.
  • User-Friendly Interface: A straightforward and intuitive web-based interface for easy input and retrieval of classification results.
  • Error Reduction: By automating the classification process, the tool helps reduce the chances of human errors in requirement categorization.

1.3 Benefits

  • Enhanced Accuracy: Improve the precision of security requirement identification, minimizing errors and potential security risks.
  • Time and Effort Savings: Streamline the software development process by automating the requirement identification and categorization tasks.
  • Comprehensive Documentation: The tool is accompanied by detailed documentation to assist users in making the most of its capabilities.
  • User-Friendly Experience: A user-centric design that ensures users can easily navigate the tool with minimal training.

2. Getting Started

2.1 Accessing the Tool

To access the Sentence Requirement Classification Tool, follow these instructions:

  • Open your preferred web browser on your computer or mobile device.
  • In the address bar, enter the http://127.0.0.1:8000.
  • Press "Enter" to navigate to the tool's login page.
  • You should now be on the main landing page of the tool, where you can log in or create a new account.

2.2 Logging In

If you already have an account, follow these steps to log in to the Sentence Requirement Classification Tool:

  • On the main landing page of the tool, locate the "Log In" or "Sign In" button. This may be prominently displayed.
  • Click on the "Log In" button to proceed to the login page.
  • On the login page, you will be prompted to enter your login credentials, which typically include your username or email address and your password. Ensure you enter these details accurately.
  • After entering your login credentials, click the "Log In" or "Sign In" button to access your account.
  • If the provided credentials are correct, you will be redirected to the tool's main interface, where you can start using its features.
  • In case you encounter any issues with login, there should be an option for account recovery or contacting support for assistance.

2.3 User Dashboard

2.3.1 Top Bar Navigation

  • Home (Current Page): Home serves as the landing page when you log in to your Sentence Requirement Classification Tool account. It is where you can input sentences for classification and receive instant results.
  • User Manual: The User Manual is a valuable resource that provides comprehensive guidance on how to use Sentence Requirement Classification Tool. It contains detailed instructions and information to help you make the most of the application's capabilities.
  • Logout Option: This option is represented as a button or link, found in the upper right corner of the interface used for session logout.

2.3.2 Using the Home Page

  • Input Form: The main component of the Home page is an input form where you can enter a sentence or requirement for classification. Simply type or paste the text into the input field.
  • Classification Button (Submit): After entering the sentence, click the "Submit" button to initiate the classification process. The system will promptly analyze the input and display the corresponding security requirement category.
  • Result Display: The classification results, including the security requirement category, will be visible on the same page.
  • Classification Tasks Navigation: Below the classification results, you'll find the "Classify Another" button. By clicking this button, you can return to the input form and initiate a new classification task.

2.3.3 Logging Out

  • Access the Logout Option: Locate the "Logout" option in the top navigation bar, typically available on all pages of the application.
  • Click the Logout Option: Click on the "Logout" button or link to initiate the logout process.
  • Session Logout: After clicking "Logout," the application will securely log you out of your account. All active sessions and data related to your session will be terminated.
  • Return to the Login Page: Following the logout process, the user will be directed to the login page.
